# Text-Based RPG Game
This is a text-based RPG game. The game features 5 rooms; you win if you can defeat all the monsters or manage to escape them.Every turn you have chance to attack escape or information.
There's a 90% chance you can escape the room and take 20 damage,or you can't escape and take 30 damage.There's a 70% chance you can attack and 40% chance you can take damage

## Features
* Choosing a character from between two classes
* Attacking the enemy
* Escaping the rooms
* Getting information about the enemy

## Technologies Used
* Python programming language
* Object Oriented Programming (OOP)
